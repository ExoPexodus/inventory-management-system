When clicking "New Entry" — how should the choice appear?

The user needs to pick between creating a Product or a Shop. Which interaction pattern feels right?

A

Split Modal

A centered dialog pops up with two large cards side-by-side. Click "Create Product" or "Create Shop" to navigate.

📦
Create Product Add a new SKU to your catalog
🏪
Create Shop Add a new store location
B

Dropdown Menu

Clicking "New Entry" opens a small dropdown beneath the button with two menu items.

📦 Create Product
🏪 Create Shop
C

Dedicated Landing Page

"New Entry" navigates to a choice page with two large tiles. From there you pick which thing to create, then go to that form.

📦
Product New SKU, variants, pricing
🏪
Shop New store location
